# 语文认字 - 学习流程重新设计

> 基于真实学习场景的流程优化

## 一、使用场景分析

### 场景1：巩固练习（学校已学）
- **背景**：孩子在学校已经学过这些字
- **目标**：回家后复习巩固，检验掌握程度
- **特点**：
  - 大部分字应该认识
  - 可能有部分字忘记了
  - 需要家长最终测试验证

### 场景2：提前预习（下学期内容）
- **背景**：孩子还没学过这些字，提前预习
- **目标**：初步学习新字，建立印象
- **特点**：
  - 很多字不认识
  - 需要先学习（看拼音）
  - 然后练习巩固
  - 最后家长测试

## 二、完整学习流程设计

```
【阶段1：学习/预习】→ 【阶段2：自主练习】→ 【阶段3：家长测试】
```

### 阶段1：学习/预习（可选）

**目的**：让孩子先看一遍所有字，建立初步印象

**功能设计**：
- **显示拼音开关**：可开可关
  - **开启**：显示字 + 拼音，适合预习场景
  - **关闭**：只显示字，适合巩固场景（跳过此阶段）
- **操作方式**：
  - 手动翻页浏览
  - 可以朗读、理解
  - 标记"已学习"（可选）
- **结束条件**：
  - 浏览完所有字
  - 或直接跳过进入练习

**UI设计**：
- 顶部显示"学习模式"标识
- 显示拼音开关（全局控制）
- 底部按钮："开始练习" / "跳过学习"

### 阶段2：自主练习（核心）

**目的**：让孩子独立认读，标记不认识的词

**功能设计**：
- **显示拼音开关**：可开可关
  - **开启**：显示拼音，帮助认读（适合预习场景）
  - **关闭**：隐藏拼音，独立认读（适合巩固场景）
- **操作方式**：
  - 手动翻页
  - 一页多词展示
  - 点击复选框标记"不认识"
  - 标记后显示拼音（帮助学习）
- **记录数据**：
  - 标记的错题
  - 练习用时
  - 查看拼音次数（如果开启拼音）

**UI设计**：
- 顶部显示"练习模式"标识
- 显示拼音开关（全局控制）
- 进度、计时等统计信息
- 卡片支持标记功能

### 阶段3：家长测试（新增，关键）

**目的**：正式权威测试，评判学习成果，作为奖惩依据

**功能设计**：
- **测试模式**：
  - 完全独立测试，**不能看拼音**（强制隐藏）
  - 家长操作，孩子认读
  - 家长标记"认识/不认识"
- **测试结果**：
  - 记录测试成绩（正确率、用时等）
  - 与自主练习结果对比
  - 生成测试报告
- **数据记录**：
  - 测试时间
  - 测试结果（正确数/错误数）
  - 测试模式标识（区分自主练习和正式测试）
  - 可作为奖惩依据

**UI设计**：
- 顶部显示"家长测试模式"标识（醒目）
- 拼音强制隐藏，不可开启
- 家长操作界面（标记对错）
- 测试完成后显示测试报告
- 与自主练习结果对比展示

## 三、流程对比

| 阶段 | 巩固场景 | 预习场景 |
|------|---------|---------|
| **学习阶段** | 关闭拼音，快速浏览（可选） | 开启拼音，认真学习 |
| **练习阶段** | 关闭拼音，独立认读 | 开启拼音，辅助认读 |
| **测试阶段** | 家长测试，验证掌握 | 家长测试，检验预习效果 |

## 四、关键功能点

### 1. 显示拼音开关
- **位置**：每个阶段都有独立的拼音开关
- **作用**：
  - 学习阶段：控制是否显示拼音学习
  - 练习阶段：控制是否显示拼音辅助
  - 测试阶段：强制关闭，不可开启
- **状态记忆**：记住用户的选择偏好

### 2. 家长测试模式
- **入口**：
  - 自主练习完成后，进入测试
  - 或从首页直接进入测试模式
- **特点**：
  - 独立的测试流程
  - 强制隐藏拼音
  - 家长操作界面
  - 记录测试结果
- **数据存储**：
  - 测试记录单独存储
  - 标记为"正式测试"
  - 可用于统计和奖惩

### 3. 结果对比
- **展示内容**：
  - 自主练习结果 vs 家长测试结果
  - 正确率对比
  - 错题对比
- **意义**：
  - 检验孩子自主练习的诚实度
  - 评估学习效果
  - 作为奖惩依据

## 六、UI/UX 改进点

### 1. 阶段标识
- 每个阶段顶部显示清晰的模式标识
- 使用不同颜色区分（学习/练习/测试）

### 2. 拼音开关
- 位置：每个阶段顶部，醒目位置
- 状态：清楚显示"显示拼音" / "隐藏拼音"
- 测试模式：显示"测试模式，拼音已锁定"

### 3. 流程导航
- 清晰的阶段进度指示
- "下一步"按钮引导流程
- 支持跳过某些阶段

### 4. 测试模式特殊设计
- 醒目的"家长测试模式"标识
- 家长操作提示
- 测试结果展示突出
