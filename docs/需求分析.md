# 语文认字 - 需求分析

> 文档同步自 `看拼音写词/docs/语文认字-需求分析.md`，后续以本文件为准。

## 1. 功能概述

- 词语导入（手动文本 + 结构化数据）
- 练习设置：题目数量、速度（秒/词）、每页词语数量、练习范围
- 练习流程：手动翻页、一页多词、手动标记错题
- 结果页：与看拼音写词一致，可修改标记
- 错题集：按轮 / 汇总视图、管理模式、批量操作
- 导入导出：词语库、练习记录、错题记录

## 2. 已确认细节

- **数据格式**：词语含 `id`、`word`、`pinyin`、`unit`；错题与练习中在标记后显示拼音
- **导入格式**：沿用看拼音写词方案，支持单元、学习园地
- **展示方式**：默认不显示拼音；横向网格布局；复用错题卡片样式
- **计时与翻页**：完全手动翻页；提供上一组 / 下一组；展示过程中即可标记
- **练习流程**：支持暂停；允许提前结束
- **结果页**：统计信息、卡片样式、手动修改标记、确认修改按钮
- **错题集**：与看拼音写词一致；管理模式下支持批量操作
- **其他功能**：从错题本练习、导入导出、命名空间前缀 `word_recognition_`
- **拼音显示逻辑**：标记错题并提交后展示拼音；错题集默认展示拼音

## 3. 实施注意事项

1. 项目放置在与「看拼音写词」并列的独立文件夹内
2. 组件暂不抽离共享库，但保持结构可迁移
3. 手动翻页时保留「上一组」「下一组」交互；速度用于参考提示
4. 暂停时停止计时，但允许继续标记
5. 导入解析将来需适配教材解析结果；当前保留 TODO 钩子
