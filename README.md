# 语文认字 - 识字练习应用

独立于「看拼音写词」的全新识字练习项目，核心功能包括：

- 词语导入：支持手动整理文本导入，后续扩展教材解析
- 练习设置：题目数量、展示速度、每页词语数量、练习范围
- 做题流程：手动翻页、一页多词、手动标记错题
- 错题管理：按轮与汇总视图、管理模式下批量操作
- 导入导出：词语库、错题记录、练习记录

## 快速开始

1. 打开 `index.html`
2. 导入词语库（支持 JSON / 文本）
3. 配置题目数量、速度、每页词语数量
4. 开始练习，使用复选框手动标记错题
